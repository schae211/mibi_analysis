Introduction

We will use MISTy to try to explain the identify of a given cell using the identify of the neighboring cells. Therefore, we one-hot encode the cell types, bypass modeling of the intraview, and only model the paraview consisting of the 50 nearest-neigbors.

expr.smp.raw <- unique(data$SampleID) %>%
  set_names() %>%
  map(function(smp) {

Computing the MISTy views. In particulat a paraview consisting of the 50 nearest neigbors (family = "constant" and l = 50)

misty.views.smp <- map2(expr.smp, pos.smp, function(expr, pos) {
  expr %>%
    create_initial_view() %>%
    add_paraview(positions = pos, l = 50, zoi = 0, family = "constant")
})
